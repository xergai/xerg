# Scripts

This directory is for task-specific scripts only.

Rules:

- name scripts clearly
- keep them small
- add a short comment at the top describing purpose and whether the script is temporary
- if a script is used more than once, either document it properly or turn it into a maintained utility
- if a script was truly one-time, delete it after the task is complete

Do not leave scratch files in the repo root. Use `tmp/` or `.tmp/` for disposable output.
